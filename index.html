Smart Copy Machine

Designed and built a gantry-style pen plotter utilizing stepper motors, driver chips, cameras, and computer vision running on a Raspberry Pi 4B+ to scan and replicate hand-drawn images by processing them into coordinate-based motor commands.

Hardware Overview

Three axes: X and Y control pen motion via belt-driven stepper motors on linear rails; Z controls the draw plate height via dual screw-driven stepper motors.

Software Overview

A camera captures the user's drawing, which is processed and skeletonized into contours, then converted into XY motor coordinates. Bresenham's algorithm handles smooth diagonal movements, with breakpoints for disconnected lines. Coordinates are translated to GPIO pulses driving MP6500 Driver Chips.

Search & Rescue Robot — "Alfred"

The goal of a robotics programming lab I took in spring 2025: program a search and rescue robot to autonomously navigate an unknown environment, find 'victims' (paper blocks), pick them up, and return them to the start. Alfred is built on a Rock Pi running ROS 2 Iron with MCUs, gyroscopes, a USB camera, two drive motors, and a two-joint arm.

Key Features

  • Autonomous Navigation: Map created in Blender with AprilTags for localization. A fixed search path is followed with onboard pathfinding for obstacle avoidance and drift correction.
  • Victim Detection & Retrieval: AprilTags on boxes enable camera-based detection. Alfred positions himself to pick up the box with his arm.
  • Return to Home: After retrieval, localization and pathfinding guide Alfred back to the start, where he drops off the box and resumes searching.

Custom Keyboard PCB

Designed and built a fully custom mechanical keyboard PCB using KiCad, inspired by the Nullbits NIBBLE kit. After my original keyboard died from a cracked PCB, I designed my own from scratch with every feature I wanted — on a custom ~70% layout.

Key Features

  • USB-C & Bluetooth: Wired or wireless via a toggle switch; onboard rechargeable battery with ~5 hours of life. MCU: Nice!Nano V2.
  • Rotary Encoders & Macros: Three programmable encoders for volume/app control; four programmable macro keys via ZMK firmware.
  • Other Features: RGB underglow, I2C OLED display, and a TRRS port for future expansion (e.g., dedicated numpad).

"Aperture Analysis"

Capstone for my Computational Physics Independent Study at Syracuse University, fall 2024. This project explored reconstructing a telescope's aperture geometry by analyzing diffraction spike patterns in stellar images. A C++ program using a custom FFT/IFFT implementation isolates diffraction spikes in the frequency domain, then uses spike angles to reconstruct the aperture shape.

Key Features

  • Custom FFT/IFFT: Fully custom implementation in C++ for frequency analysis and diffraction spike isolation.
  • Spike Analysis: Analyzed angles and intensities of diffraction spikes to infer aperture geometry.
  • Visualization: Data piped into Python/Matplotlib to render a 2D representation of the reconstructed aperture.
